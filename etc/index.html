HPC Twitter Dumps

HPC Twitter Dumps allows to download the 10% of the daily stream of tweets produced on Twitter. The tweets are a random sample of the tweets in Twitter and are downloaded thanks to the Garden Hose Streaming API provided by Twitter.

We would like to share the tweets with all the CNR researchers interrested, but since we signed an agreement regarding the data usage policy, we kindly ask you to send us an email to let us know that you are using the data and with what intention.

Available dumps (repositories)

We split the tweets in one file per day, each file is encoded in JSON. More in detail, each line of a file represents a tweet encoded in JSON. You can read the file line per line, decoding each line from JSON to an object representing the tweet (Please note that there are libraries for doing that, you don't have to write your own parser).

Every day we collect all the 10% of the tweets provided by the streaming API in on unique file, the compressed file size is ~20G and it contains approx 40M of tweets

Since you probably don't want all the tweets, we provide others filtered dumps which probably match your needs. If you need other filters please let us know.

Please do not download the full dump files because the download could interfere with the current download from the streaming API or stop the process. If you need the full dump, please contact us.
